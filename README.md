# Tennis Game
1. It is a single-player game, wherein the left player is the user and the right player is the computer.
2. The left player paddle can be changed using the mouse as per the ball movement. 
3. The score is displayed on the board. 
4. If a player misses a ball, the other player gets a point. The player that reaches three points first wins the game.
5. After a point is scored, game control resets and starts from the center.
6. In the end, the winner is displayed on the board.
